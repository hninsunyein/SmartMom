# Smart Mom — Next.js Frontend

This is the frontend for the Smart Mom Balancing System, built with **Next.js 15** (App Router), **Tailwind CSS v4**, and **Redux Toolkit**.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **State Management**: Redux Toolkit
- **HTTP Client**: Axios
- **Icons**: Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run start` | Start production server |
| `npm run lint` | Lint source files |

## Project Structure

```
src/
├── app/           # Next.js App Router pages
├── components/    # React components (auth, dashboard, shared)
├── redux/         # Store, slices, provider
└── services/      # API service layer
```
